What is Tau Labs

Tau Labs is an Open Source project aimed to create high quality code for controlling multirotors and other UAVs. Please see our about page for a long description of our goals objectives. For a list of features of our software, please see this page. The developement is driven entirely by volunteers donating their time in pursuit of this goal. Tau Labs also has an active community and we invite you to drop into our IRC channel (freenode.net #taulabs) or forums.
